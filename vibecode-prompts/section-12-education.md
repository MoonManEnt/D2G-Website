# Section 12: Education Hub

## Learn Screen Layout

### Header
- Title: "Learn"
- Search bar
- Progress indicator: "You've completed X of Y lessons"

### Categories
- **Credit Basics** — How credit works
- **Your Rights** — FCRA, FDCPA explained
- **Disputing 101** — How to dispute effectively
- **Building Credit** — From scratch or rebuilding
- **Avoiding Scams** — Protect yourself
- **Advanced Strategies** — For experienced users

### Content Types
1. **Video Explainers** (1-2 minutes each)
2. **Articles** (plain language, scannable)
3. **Interactive Quizzes**
4. **Infographics**

## Lesson Format

### Video Lesson
- Thumbnail with play button
- Duration shown
- Transcript available
- Mark as complete button

### Article Lesson
- Clean reading layout
- Progress bar as you scroll
- Key takeaways box at end
- "Was this helpful?" feedback

### Quiz
- Multiple choice questions
- Immediate feedback (right/wrong)
- Explanation for each answer
- Score at end
- Retry option
