# Section 4: Dashboard & Credit DNA

## Dashboard Layout (Home Screen)

### Header
- Greeting: "Hey [First Name]" or "Good morning, [First Name]"
- Current date
- Settings gear icon (top right)

### Credit Score Display (Primary Focus)
- Large orbital/arc visualization
- Center: Average score across bureaus (large number)
- Arc shows score position (color-coded)
- Below arc: Individual bureau scores
  - TU: XXX | EX: XXX | EQ: XXX
  - Each color-coded by score range
- Tap to expand → Score details screen

### Credit DNA Card
- Headline: "Your Credit DNA"
- Classification badge with emoji:
  - Recovering
  - Thin File Rebuilder
  - Collection Heavy
  - Late Payment Pattern
  - High Utilization
  - Inquiry Damaged
  - Identity Issues
  - Near Prime
- One-line description of what this means
- Tap to expand → Full DNA analysis

### Quick Stats Row (4 cards)
- **Accounts:** Total count
- **Negative:** Count (amber/red highlight if >0)
- **Disputes:** Active count
- **Days:** Until next action needed

### Recommended Action Card
- Icon: Target
- Headline: "Recommended Next Step"
- AI-generated suggestion based on their profile
- Example: "Dispute that Midland collection first — it's your highest impact item."
- Button: "Start Dispute →"

### Progress Timeline (if disputes active)
- Horizontal timeline showing journey
- Milestones: Report Uploaded → First Dispute → Response Received → Item Removed
- Current position highlighted

### Floating Amelia Button
- Bottom right corner
- Chat bubble icon
- Tap to open AI assistant

## Credit DNA Analysis (Expanded View)

### Classification System
Each user gets ONE primary classification:

| Classification | Criteria | Description |
|----------------|----------|-------------|
| Recovering | Past damage + recent positive activity | "Past damage with positive momentum" |
| Thin File Rebuilder | <5 accounts, limited history | "Building credit from scratch" |
| Collection Heavy | 3+ collection accounts | "Multiple collections weighing you down" |
| Late Payment Pattern | 4+ late payments across accounts | "Payment timing needs work" |
| High Utilization | >50% average utilization | "Credit cards are maxed out" |
| Inquiry Damaged | 6+ hard inquiries in 12 months | "Too many recent applications" |
| Identity Issues | Accounts flagged as potentially fraudulent | "Possible identity theft detected" |
| Near Prime | Score 650-699, minor issues | "Almost there, just a few fixes needed" |

### Health Scores (0-100 scale)
Display as progress rings:
- **Overall Health:** Composite score
- **Improvement Potential:** How much room to grow
- **Urgency:** How quickly to act

### Key Insights
- 3-5 bullet points in plain language
- Example: "That $2,340 Midland collection is hurting you the most"
- Example: "Your oldest account is helping your score — keep it open"

### Immediate Actions
- Numbered list of 3 recommended actions
- Each links to relevant feature (dispute, education, etc.)
